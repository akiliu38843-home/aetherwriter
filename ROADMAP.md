# 🚀 AetherWriter 产品迭代路线图

基于当前产品形态和定位，制定详细的功能迭代方向。

---

## 🎯 产品定位回顾

**一句话定位**：基于热内特叙事学的专业 AI 小说润色工具

**核心价值**：
- 不是写作助手，而是**文字打磨专家**
- 专注于叙事维度控制（距离、聚焦、时间）
- 适用于日本轻小说、推理小说、纯文学等多种类型

**目标用户**：
- 网络小说作者
- 文学创作爱好者
- 专业作家（润色工具）

## 💡 具体功能建议

### 1. **"一键润色"功能**

**场景**：用户选中一段文字，希望快速润色

**建议实现**：
```typescript
// 基于预设的"平衡模式"自动润色
const quickPolish = () => {
  const dimensions = {
    distance: 'near',
    focalization: 'internal',
    duration: 'scene'
  }
  handleNarrativeRewrite(dimensions)
}
```

**用户体验**：
- 选中文字 → 右键菜单 → "一键润色"
- 或快捷键 `Ctrl + Shift + P`
- 自动使用默认配置润色

---

### 2. **"润色模式"预设**

**建议预设模式**：
1. **轻小说模式**
   - 距离：较近
   - 聚焦：内聚焦
   - 时间：心理场景
   - 特点：清新、自然对话

2. **推理小说模式**
   - 距离：较远
   - 聚焦：外聚焦
   - 时间：叙事跨越
   - 特点：冷峻、客观

3. **纯文学模式**
   - 距离：极近
   - 聚焦：零聚焦
   - 时间：审美停顿
   - 特点：诗意、意象丰富

4. **网文爽文模式**
   - 距离：极近
   - 聚焦：内聚焦
   - 时间：叙事跨越
   - 特点：节奏快、高潮多

---

### 3. **"AI 续写"功能**

**场景**：用户写到一个情节高潮，希望 AI 给出后续多个可能的发展方向

**建议实现**：
```typescript
const handleContinue = async () => {
  // 获取当前选中文本的前后文
  const context = editor.getText()
  
  // 生成 3-5 个续写方向
  const continuations = await generateContinuations(context, {
    count: 3,
    directions: ['常规发展', '意外转折', '深化冲突']
  })
  
  // 显示在侧边栏，用户选择后插入
  showContinuationsPanel(continuations)
}
```

### 4. **"章节摘要"功能**

**场景**：用户写了多章内容，希望快速生成章节摘要

**建议实现**：
```typescript
const generateChapterSummary = async (chapterContent: string) => {
  const summary = await ai.summarize(chapterContent, {
    style: 'narrative',
    maxLength: 200
  })
  return summary
}
```

**用途**：
- 生成章节目录
- 辅助大纲管理
- 快速回顾内容
